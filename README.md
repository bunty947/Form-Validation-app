# Form-Validation-app
A simple full‑stack application for user registration and form validation.
Frontend: HTML,CSS, Javascript(ODM)
Backend: Node.js, Express.js
Database: MongoDB (with Mongoose ODM)
Templating Engine: EJS
NPM pcakages: Method-override, nodemon (for devlopment)

Features
User registration form with client-side validation (email, password, gender and age-check)
Password toggle visibility and real-time validation feedbak( if correct then border of box will be in green else it will bein red color)
CRUD operations (Create Read Update Delete) for users
Server-side validation with Mongoose schema.
EJS templates to render the user list, display specific user's profile and edit/modify it.
RESTful routes with method-override for PUT/PATCH/DELETE.

How to run
Install dependenices from npm
npm i express
npm i ejs
npm i method-override
npm i nodemon
